### :shopping_cart: **marketplace-orchestrator** — Multi-Vendor E-Commerce Platform

- Real-time inventory synchronization across multiple vendors
- Dynamic pricing engine with rule-based composition
- Live order tracking with WebSocket notifications
- Seller analytics dashboard with Recharts visualizations

### :dart: **sentinel-stream** — Distributed Object Detection Framework
**Tech Stack:** Python · OpenCV · YOLO · NumPy
- Multi-camera real-time surveillance with threat assessment
- Distributed processing pipeline for concurrent video streams
- Configurable detection zones and alert thresholds
- Type-safe module architecture with comprehensive error handling

### :chart_with_downwards_trend: **pricepulse** — Price Drop Alert System
**Tech Stack:** Next.js 14 · TypeScript · Prisma · PostgreSQL · Stripe · Chart.js
- Automated price monitoring with customizable alert rules
- Historical price tracking with interactive charts
- User notification system for price drops
- Subscription management with Stripe integration

### :brain: **churn-risk-scorer** — ML Customer Churn Predictor
**Tech Stack:** Python · scikit-learn · Pandas · NumPy · Plotly
- Customer behavior analysis with feature engineering
- Multiple ML model comparison (Random Forest, XGBoost, Logistic Regression)
- Interactive visualization dashboard with Plotly
- Production-ready prediction API with comprehensive test suite

### :grinning: **face-mood-detector** — Real-Time Emotion Recognition
**Tech Stack:** Python · TensorFlow · OpenCV · Keras
- CNN-based deep learning model for facial emotion classification
- Real-time webcam integration with live predictions
- Support for 7 emotion categories
- Trained on FER-2013 dataset with data augmentation

### :lotus_position: **Tranquility** — Music & Wellness App

- Meditation timer with guided breathing exercises
- Responsive design with smooth animations
- Offline-first architecture for uninterrupted sessions
